Hackathon Attendee
Safe Dsumption is a website that allows people to search for the weather of specific location. People can check the UV Index for specific city and state and the website will display the instruction how to protect themselves from sunburn. The quiz is used to evaluate the risks of skin cancer and vitamin D deficiency. It will help you to decide if you should get sunlight for vitamin or you should avoid sunlight. The project was named as "Safe Dsumption" because I want to raise awareness the damages of sunbathing. People need to consume vitamins D but they need to do it safety. I had a great time learning about OpenWeatherMap API for the first time. You can view my project at https://devpost.com/software/safe-dsumption
